## Características y Funcionalidades principales
- Diseño responsive (Mobile First) utilizando HTML, CSS y JavaScript
- HTML semántico y accesible
- CSS moderno (Flexbox, Grid, Media Queries)
- Contiene animaciones al desplazarse
- Incluye un formulario para calcular el índice de masa corporal.
- Desarrollado primero con la metodología Mobile First, luego para PC de escritorio.
- Secciones informativas (planes, servicios)
